Commands

Music Commands Documentation

The ParadoxPure Bot offers a variety of music commands to enhance your music listening experience in your server. The prefix for all music commands is a comma (,). Below is a list of available music commands along with their explanations:

  • Play: play
    The play command allows you to play a song in your voice channel. You can specify the name of the song or provide a YouTube URL as the argument. The bot will search for the song and add it to the queue.
  • CQueue: cqueue
    The cqueue command displays the current queue of songs in a compact format. It shows the upcoming songs in a concise list.
  • Join: join
    The join command makes the bot join the voice channel you are currently in. This is necessary for the bot to play music in your server.
  • Jump: jump
    The jump command allows you to skip to a specific position in the queue. Provide the position number as the argument to jump directly to that song in the queue.
  • Loop: loop
    The loop command toggles the looping of the currently playing song. When looped, the current song will repeat indefinitely until you disable the loop.
  • Pause: pause
    The pause command pauses the currently playing song. You can use the resume command to continue playing the paused song.
  • Queue: queue
    The queue command displays the current queue of songs. It shows the upcoming songs and their order in the queue.
  • Radio: radio
    The radio command plays a radio station of your choice. Specify the radio station as the argument, and the bot will search for it and start playing the station.
  • Remove: remove
    The remove command allows you to remove a specific song from the queue. Provide the position number of the song as the argument to remove it from the queue.
  • Repeat: repeat
    The repeat command toggles the repeating of the entire queue. When enabled, the entire queue will repeat once it finishes playing.
  • Seek: seek
    The seek command allows you to seek to a specific time in the currently playing song. Provide the time in the format HH:MM:SS or MM:SS.
  • Shuffle: shuffle
    The shuffle command shuffles the songs in the queue, changing their order randomly. This provides a randomized listening experience.
  • Skip: skip
    The skip command allows you to skip the currently playing song and move to the next song in the queue.
  • Spotify: spotify
    The spotify command allows you to play music directly from a Spotify playlist, album, or track. Provide the corresponding URL as the argument.
  • Stop: stop
    The stop command stops the playback of the current song and clears the queue.
  • Volume: volume
    The volume command allows you to adjust the volume of the bot. Specify the desired volume percentage as the argument.

Other Commands Documentation

The ParadoxPure Bot offers a variety of music commands to enhance your music listening experience in your server. The prefix for all music commands is a comma (,). Below is a list of available music commands along with their explanations:

  • Invite: invite
    The invite command provides you with the invite link for adding Wick Bot to your server.
  • Vote: vote
    The vote command provides you with the link to vote for Wick Bot on the bot listing website.
  • Help: help
    The help command displays the list of available music commands and their explanations.
  • Ping: ping
    The ping command measures the bot's latency or response time to Discord.
  • Avatar: avatar [user mention]
    The avatar command displays the avatar or profile picture of a specified user. If no user is mentioned, it shows your own avatar.
  • Banner: banner [user mention]
    The banner command displays the banner or profile cover image of a specified user. If no user is mentioned, it shows your own banner.
  • ServerAvatar: serveravatar
    The serveravatar command displays the avatar or server icon of the current server.
  • Stats: stats
    The stats command displays the statistics and information about Wick Bot.
  • MemberCount: membercount
    The membercount command shows the total number of members in the current server.
